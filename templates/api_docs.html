DNS Lookup API Documentation

Endpoint: /api/v1/dns-lookup

Method: POST

Request Body:

{
    "dns_name": "example.com",
    "dns_type": "A",
    "dns_servers": ["8.8.8.8", "1.1.1.1"]
}

Response:

{
    "dns_name": "example.com",
    "dns_type": "A",
    "results": {
        "8.8.8.8": ["93.184.216.34"],
        "1.1.1.1": ["93.184.216.34"]
    }
}
